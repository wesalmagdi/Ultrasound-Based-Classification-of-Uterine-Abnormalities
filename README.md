# Ultrasound-Based Classification of Uterine Abnormalities

## 📌 Overview
This project focuses on building a deep learning model to classify uterine abnormalities using ultrasound images. The goal is to assist in medical diagnosis by leveraging computer vision techniques to improve accuracy and consistency.

---

## 🎯 Objectives
- Apply deep learning techniques to medical imaging
- Classify ultrasound images into meaningful diagnostic categories
- Evaluate model performance using appropriate metrics

---

## 🧠 Methodology
1. **Data Preprocessing**
   - Image cleaning and resizing
   - Normalization
   - Data augmentation (if applied)

2. **Model Development**
   - Built and trained deep learning models for classification
   - Experimented with architectures and hyperparameters

3. **Evaluation**
   - Used performance metrics such as accuracy / F1-score
   - Analyzed results and model behavior
